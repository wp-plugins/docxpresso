Docxpresso inserts content from a document file (.odt).

== Description ==

This plugin will allow you to publish content that has been generated with your
favourite Office Suite (MS, libre or Open Office).

Before uploading the documents you should save them with the .odt extension
(Open Document Format) that is available in practically any standard Office 
Suite (MS Office included).

== Installation ==

1. Put the plugin folder into [wordpress_dir]/wp-content/plugins/ or use the
add plugin option of the Wordpress admin interface.
2. Go into the WordPress admin interface and activate the plugin
